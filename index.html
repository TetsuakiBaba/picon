Picon

Picon is a parametric SVG icon generator that can be easily used with just html tags, such as i, svg or js.

icons

clock
parameters

Quick start

<!doctype html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>picon simple sample</title>
</head>

<body>
    <i data-pc data-pc-name="clock" data-pc-hour="10" data-pc-minute="10"></i>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/svg.js/3.1.2/svg.min.js"

        crossorigin="anonymous" referrerpolicy="no-referrer"></script>
    <script src="./picon.js" crossorigin="anonymous" type="text/javascript"></script>
</body>

</html>
