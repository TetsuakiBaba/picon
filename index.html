Picon

Picon is a parametric SVG icon generator that can be easily used with just html tags, such as i, svg or js.

Via JavaScript

Create a picon with a single line of JavaScript; 

                new picon('#svg', 'clock', {
    clock: {
        hour: 10,
        minute: 8,
    }
});

Options

name parameter(s)
clock hour, minute
battery percentage(0-100)
calendar text(Up to 2 letters are beautiful in appearance. 1 character is better for 2-byte characters)

Dependency

SVG.js: https://github.com/svgdotjs/svg.js
